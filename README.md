# InterviewAI

AI-powered interview preparation platform built with modern web technologies to help users prepare for technical and behavioral interviews.

## Features

### Current Functionality

- **Technical Interview Prompts**: AI-generated technical questions tailored to various domains
- **Behavioral Interview Prompts**: Role-specific behavioral and situational questions
- **Intelligent Scoring System**: Real-time evaluation and feedback
- **Session Memory**: Persistent session tracking and history
- **WebSocket Support**: Real-time communication between frontend and backend
- **Interactive UI**: Responsive user interface for seamless practice

## Architecture

### Backend
- Node.js/Express server with WebSocket support
- Real-time API endpoints for interview sessions
- Scoring and evaluation engine
- Session management

### Frontend
- Modern responsive UI for interview practice
- Real-time communication with backend
- Interactive question and answer interface
- Performance analytics

## Getting Started

### Prerequisites
- Node.js v14+ installed
- npm or yarn package manager

### Installation

#### Backend
```bash
cd backend
npm install
```

#### Frontend
Refer to the frontend directory documentation for setup instructions.

### Running the Application

#### Backend Server
```bash
cd backend
npm start
```
**Server runs at**: http://localhost:8000

#### Frontend
Follow the frontend directory documentation to run the frontend application.
